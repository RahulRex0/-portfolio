# Rahul Rex — Portfolio

A personal portfolio website built to showcase my projects, skills, and learning progress.

The website contains interactive and visually organized details about the projects I have worked on, along with up-to-date information about me.

## Tech Stack used on current website

- Next.js

- TypeScript

- Tailwind CSS

- JavaScript

## Features

- Personal introduction section

- Project listing page

- Individual project detail pages

- Responsive design for desktop and mobile

- Clean and simple UI
